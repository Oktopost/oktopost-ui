# Search Form
Use this component to display a simple search form.

## Usage 
JavaScript:

```JavaScript
var SearchForm = window.OUI.components.SearchForm;

var sFrom = new SearchForm('div.search-form-wrapper', 'Search');

```

HTML:

```HTML
<div class="search-form-wrapper"></div>
```

## Events
1. onKeyup
2. onKeydown
3. onChange
4. onClear
